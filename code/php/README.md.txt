PROJECT NAME : SORTING PROCCESS OF FICTIONARY FILM STREAMING WEBSITE

PROJECT DESCRIPTION: 

Input of 10 movies, each having a different array rerpresenting the title, year of production, genre and the rating

REQUIRED TASK:
Transform the rating from percentage to stars (0.5 -> 5 stars)
Sort the list of movies based on different criterias (titles, year, genre or rating)

FOUND SOLUTION:
Create function that uses iterativity to change the rating from percentage to stars
Create new array in associative manner o contain all the data of the 4 arrays
Create function that sorts the array based on the specified parameter

HOW TO:
The program runs in the console. There is no HTML/CSS implemented. 
To change the filter of the sorting process you need to change the second parameter in the callinf of function movieSort().
In the same function, you can also change if you want it to be ordered in ascending
or descending order.
